Running the code requires jython. Once the ABAGAIL.jar library has been built,
change directories into random_optimization/ABAGAIL/jython and run the python scripts
```jython countones.py```
```jython fourpeaks.py```
```jython travelingsalesman.py```
```jython abalone_test2.py```

These scripts do not take arguments. 
Data was written to stdout and was manually added to a spreadsheet to produce charts.
